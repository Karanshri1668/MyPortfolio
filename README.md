# 🚀 Developer Portfolio
A modern developer portfolio designed to showcase projects, technical skills, certifications, and professional experience through an interactive and responsive interface.

The portfolio automatically fetches repositories using the GitHub API and presents them with filtering, searching, sorting, and featured project functionality.

## ✨ Features

- Modern responsive UI
- GitHub API integration
- Featured project showcase
- Dynamic project filtering
- Search & sorting
- Interactive timeline
- Certificate carousel
- Resume integration
- Glassmorphism UI
- GSAP animations
- Mobile-first responsive design

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub REST API
- GSAP
- Font Awesome
